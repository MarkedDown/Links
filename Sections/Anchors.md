
# Anchors

*Anchors can be used to link to sections **inside** your files.*
 
<br>

## Headlines

*Headlines are a type of anchor and* <br>
*can be linked to in the following way:*

<br>

```markdown
## Some Headline

[Link To Some Headline]: #some-headline
```

<br>
<br>
